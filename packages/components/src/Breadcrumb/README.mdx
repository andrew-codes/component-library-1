## API

### Breadcrumb

<PropsTable of={Breadcrumb} />

### Breadcrumb.Item

| Property | Type | Required | Default  | Description                                         |
| :------- | :--- | :------- | :------- | :-------------------------------------------------- |
| children | Node | false    |          | Link, anchor, or text                               |
| selected | bool | false    | false    | If true the Breadcrumb item is selected             |
| onClick  | func | false    | () => {} | Function called when the Breadcrumb item is clicked |
